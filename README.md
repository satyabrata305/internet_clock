Here’s a complete, runnable ESP-IDF C program that implements an Internet Clock using NTP (SNTP) to fetch accurate time from the internet and display it via the serial monitor or oled display.

This works on ESP32 with Wi-Fi and ESP-IDF (v4.x or v5.x).

How It Works:-
This project is an wifi based internet clock,which connects to internet through wifi router and fetches the time information 
using NTP & displays on oled display.



Dev kit - ESP32 wroom

IDE - Espressif (ESP-IDF)

Language - Embedded C
